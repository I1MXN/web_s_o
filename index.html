¿Qué es HTML?

Es el lenguaje utilizado para estructurar el contenido de las páginas web mediante etiquetas que definen elementos como textos, imágenes y enlaces.

¿Qué es CSS?

Es un lenguaje que define el estilo y diseño visual de las páginas web, como colores, fuentes y el diseño de los elementos.

¿Qué es JavaScript?

Es un lenguaje de programación usado para crear interactividad en páginas web, como animaciones, formularios dinámicos y manejo de eventos.
